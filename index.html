Contexto del Problema

El proyecto consiste en una plataforma web interactiva diseñada para el contexto boliviano, que permite a los usuarios simular y comprender el impacto socioeconómico de situaciones críticas como el desabastecimiento de carburantes, la inflación de precios, el impacto de bloqueos sociales y la escasez de recursos; mediante una aplicación web educativa basada en modelos matemáticos sencillos, el sistema democratiza el análisis de datos financieros y de consumo, facilitando la visualización de resultados dinámicos y promoviendo una toma de decisiones informada y resiliente ante la incertidumbre del entorno

Conclusiones

Modelos matemáticos simples son poderosos

Fórmulas básicas permiten visualizar tendencias complejas de manera comprensible.

JavaScript y DOM en la práctica

Este proyecto demostró cómo capturar datos y renderizar resultados dinámicos sin recargar la página.

Diseño responsivo es esencial

Adaptar la interfaz a celulares y computadoras garantiza accesibilidad para todos.

Utilidad comunitaria real

Una familia puede usar este simulador para planificar sus compras y tomar mejores decisiones.
